Installation

Supported Platforms

SQLift provides native executables for the following platforms:

  • Linux (x64)
  • macOS (x64, arm64)
  • Windows (x64)

Quick Install

Install SQLift using curl:

curl -fsSL https://sqlift.dev/install.sh | sh

Manual Installation

  1. Download the appropriate binary for your platform from the releases page
  2. Extract the archive
  3. Add the binary to your PATH

Verifying Installation

Verify the installation by running:

sqlift --version

Getting Started

After installation, you can start using SQLift with these basic commands:

1. Initialize a new project

sqlift init

2. Configure your project

Create or modify sqlift.yaml:

version: "1"
sql:
  engine: "postgresql"
  schema: "schema.sql"
  output:
    package: "com.example.entity"
    options:
      lombok: false
      jpa:
        enabled: true
        type: "jakarta"

3. Generate entities

sqlift generate
